General Info

You use this program by importing your Collection from DVD Profiler via the XML Export function (DVD Profiler -> File -> Export Profile Database).
For the export you can uncheck all the optional export data (e.g.cast and crew), since it is of no relevance for this task. Then you import it here.
The program distinguishes between profiles with an UPC/EAN and manual or DiscId profiles.
Once you enter the UPC/EAN of a profile in the text box it will be found and removed from the list. What remains are your missing DVDs.
Manual or DiscId profiles have to be removed by hand.

Startup Parameters

/skipversioncheck

If you start the program with the parameter /skipversioncheck then it won't check if a new version is available for download.
